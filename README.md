# WindowsFormsApp1
To log in, use this default user, username- kabbo, pass- admin123 this account is a super user, with the ability to create other admin accounts and modify them.
Database Setup:

Open SQL Server Management Studio (SSMS).

Open the BeautyStoreDB.sql script included in the root directory.

Execute the script to generate the database schema and seed data.

Configure Connection:

Open the project in Visual Studio.

Navigate to Data/DatabaseConnection.cs.

Update the connectionString variable to match your local SQL Server instance name (e.g., Server=YOUR_PC_NAME\\SQLEXPRESS).

Build & Run:

Press F5 or click Start in Visual Studio to compile and launch the application.
